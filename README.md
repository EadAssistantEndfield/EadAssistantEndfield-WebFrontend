# Blueprint Analysis

这个目录是独立的 `Vite + Vue 3 + TypeScript` 蓝图分析前端。

## 启动

```bash
cd blueprint_analysis
npm install
npm run dev
```

默认开发地址：

```text
http://127.0.0.1:4173
```

## 功能

- 拖拽或上传本地蓝图 `json`
- 直接粘贴 JSON 并重新解析
- 渲染蓝图概览、建筑统计、平面布局、节点清单
- 使用 Vue 3 Composition API + `<script setup>`

## 构建

```bash
npm run build
npm run preview
```
